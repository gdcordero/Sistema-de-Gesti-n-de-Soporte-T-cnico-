🛠️ Sistema de Gestión de Soporte Técnico

Diseño e implementación de base de datos relacional en MySQL Workbench. Proyecto académico en desarrollo (Etapa 2 de 4).

📋Requerimientos de Negocio (Fase 1)
El diseño inicial se centró en la capacidad de la empresa para:
Reportes de Solicitudes: Capacidad de agrupar solicitudes por estado (Pendiente, En proceso, Finalizado).
Historial Cronológico: Visualización de eventos de una solicitud ordenados por fecha.
Trazabilidad Técnica: Identificación de los técnicos involucrados en cada resolución.

💻Implementación Técnica (Fase 2)
En esta etapa, se desarrollaron sentencias SQL avanzadas para satisfacer los siguientes requerimientos del I Cuatrimestre 2026:
Gestión de Datos: Comandos para limpieza y carga masiva de datos según el diagrama E-R.
Consulta de Órdenes de Servicio: Generación de información detallada que integra usuarios, técnicos, estados de servicio y detalles del equipo (número de serie y nombre).
Métricas por Estado: Consultas de agregación para cuantificar órdenes de servicio según su tipo de estado.
Actualización Masiva: Comandos SQL para la gestión de estados automáticos basados en fechas de completado no nulas.

⚙️Tecnologías Utilizadas
Motor: MySQL Server 8.0.
Entorno: MySQL Workbench.
Formación: Diplomado en Ingeniería Informática, UNED.
